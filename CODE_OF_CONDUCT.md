# Code of Conduct

This project follows the
[CNCF Community Code of Conduct](https://github.com/cncf/foundation/blob/main/code-of-conduct.md).

By participating in this project you agree to abide by its terms. Adopting the
CNCF Code of Conduct keeps the project aligned with the expectations of a
neutral, vendor-independent home.

## Reporting

To report a violation, contact the maintainers listed in
[`MAINTAINERS.md`](MAINTAINERS.md). Reports will be handled confidentially.
